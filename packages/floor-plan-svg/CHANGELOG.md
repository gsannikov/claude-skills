# Changelog

All notable changes to the floor-plan-svg skill.

## [0.1.0] - 2025-12-12

### Added
- Initial skill structure
- SKILL.md with commands and workflow
- Input parser module (PDF, DXF, image, spec)
- SVG generator module with svgwrite
- Fixture library (toilet, vanity, shower, bathtub, door, window)
- Annotation engine (dimensions, compass, scale bar)
- Scale configuration (1:20, 1:50, 1:100, 1:200)
- Python script for programmatic generation
- Detailed workflow reference
- Path configuration

### Technical
- svgwrite-based SVG generation
- ezdxf for DXF parsing support
- PyMuPDF for PDF extraction support
- Vision-based photo analysis support
