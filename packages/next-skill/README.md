# Next Skill - Skill Factory

AI-assisted skill creation with GitHub discovery, adaptation, or fresh scaffolding.

## Capabilities

- **Discovery**: Search GitHub and MCP registries for existing solutions.
- **Adaptation**: Refactor external code to Exocortex architecture.
- **Scaffolding**: Generate new skills from standard templates.
- **Validation**: Ensure skills comply with repo standards.

## Usage

```bash
# Create a new skill (interactive workflow)
/create skill: [description]

# Search for existing solutions
/search skills: [query]

# Scaffold a fresh skill directly
/scaffold: [name]
```
