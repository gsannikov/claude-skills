# Social Media Post Generator

Generate platform-optimized social media posts using algorithm insights and best practices.

## Quick Start

1. Tell Claude: `"Create a Threads post about [topic]"`
2. Specify platform: Threads, X (Twitter), or LinkedIn
3. Choose style: short & punchy, medium detailed, or long-form

## Commands

| Command | Action |
|---------|--------|
| `Create Threads post about [topic]` | Generate Threads-optimized post |
| `Write X post for [feature]` | Generate Twitter-optimized tweet |
| `Create LinkedIn announcement for [release]` | Generate professional LinkedIn post |
| `Generate social media posts for [announcement]` | Multi-platform campaign |

## Features

- **Platform Optimization**: Character limits, formatting, hashtag strategies
- **Algorithm Insights**: 2025 algorithm priorities for each platform
- **Post Variants**: Short (X-compatible), Medium (Threads), Long-form (LinkedIn)
- **Engagement Scoring**: 1-10 scale based on hook strength, value clarity, CTAs
- **Posting Times**: Optimal times for US audience by platform
- **Templates**: Product launches, feature announcements, technical threads

## Platforms

| Platform | Char Limit | Hashtags | Key Focus |
|----------|------------|----------|-----------|
| Threads | 500 (10K with attachment) | None | Conversational, questions |
| X | 280 (25K Blue) | 1-2 max | Concise, visual, threads |
| LinkedIn | 3,000 | 3-5 max | Professional, data-driven |
