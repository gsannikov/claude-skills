## [1.0.0] - 2025-11-27

### Added
- Initial release
- Platform-specific post generation for Threads, X (Twitter), and LinkedIn
- Content analysis with `post_analyzer.py` script
- Platform algorithm insights and best practices
- Multiple post variants (short, medium, long-form)
- Character count validation and metadata
- Templates for product launches, feature announcements, and technical deep-dives
- Engagement scoring system (1-10 scale)
- Optimal posting time recommendations

### Platforms Supported
- Threads (Instagram) - Conversational, no hashtags
- X (Twitter) - Concise, 1-2 hashtags, threads
- LinkedIn - Professional, data-driven
