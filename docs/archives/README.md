# Archives

This folder contains historical documentation, session reports, and analysis from development work.

## Contents

### Session Reports

Session reports document specific development work, bug fixes, and improvements:

- **INCONSISTENCY_REPORT.md** - Analysis of repository inconsistencies from incomplete migration (Nov 2025)
- **FIXES_SUMMARY.md** - Summary of all fixes applied to resolve inconsistencies (Nov 2025)
- **WORKFLOW_IMPROVEMENTS.md** - Comprehensive recommendations for GitHub Actions workflow improvements (Nov 2025)

### Purpose

These files provide:
- Historical context for major changes
- Detailed analysis for future reference
- Learning material for contributors
- Audit trail of development decisions

---

## Guidelines for This Folder

**What belongs here:**
- Session/development reports
- Analysis documents
- Historical summaries
- Deprecated documentation
- Old versions of files

**What doesn't belong here:**
- Active documentation (goes in appropriate docs/ subfolder)
- Code (goes in appropriate source folder)
- Configuration (goes in config/)

---

**Note:** Files in this folder are for reference only and may not reflect the current state of the repository.
